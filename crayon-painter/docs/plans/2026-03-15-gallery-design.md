# 画展与画材记录功能设计

## 概述

在现有蜡笔绘画应用中添加画展功能和画材自动记录功能。用户绘画时自动拍摄画材照片，保存时可命名并展示在个人画展中。

## 功能需求

### 1. 画材自动记录

- 绘画过程中，当检测到物体颜色发生明显变化时（换了个物体），自动从摄像头截取当前画面作为"画材"照片
- 保存画材的颜色信息用于显示
- 最多记录 10 个画材，避免存储过多

### 2. 保存流程

点击 SAVE 按钮后：
- 弹出浮层展示画作预览图
- 显示已记录的画材缩略图列表
- 输入框供用户命名
- "保存"按钮 - 保存到本地存储
- "查看画展"按钮 - 跳转到画展页面

### 3. 画展页面

- 日式简约设计风格，大量留白
- 网格或列表形式展示所有保存的画作
- 每张画显示：
  - 画作缩略图
  - 用户命名的名字
  - 创建日期
  - 画材小图标列表（颜色圆点）
- 点击可查看大图和详情

## 技术方案

### 数据结构

```javascript
// localStorage key: 'wuse_artworks'
{
  artworks: [
    {
      id: timestamp,
      name: "用户命名的名字",
      createdAt: "2026-03-15 14:30",
      imageData: "data:image/png;base64,...",
      materials: [
        { color: {r: 255, g: 100, b: 50}, thumbnail: "data:image/png;base64,..." }
      ]
    }
  ]
}
```

### 存储策略

- 使用 localStorage 存储
- 图片转换为 base64 存储
- 超过 5MB 时提示用户删除旧作品

## 页面设计

### 保存弹窗

- 半透明黑色背景遮罩
- 居中白色卡片
- 顶部：画作预览
- 中部：画材缩略图横排（颜色圆点 + 照片）
- 底部：输入框 + 两个按钮

### 画展页面

- 顶部：标题 "我的画展"
- 网格布局展示作品卡片
- 卡片：图片 + 名字 + 日期 + 画材色点
- 返回按钮

## 验收标准

1. 绘画时更换物体颜色能自动记录画材
2. SAVE 弹窗正确显示画作和画材
3. 可以命名并保存成功
4. 画展页面正确展示所有作品
5. 数据持久化有效（刷新页面仍在）
